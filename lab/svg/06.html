SVG Demo - 06

Illustrating how to include an <a> in an SVG image.

Note the background colors are purely to help illustrate the size of the links and the hover effect.

Highlighted code is code that has been added or changed since the previous demo.

HTML:

<object type="image/svg+xml" data="img/fallback-logo.svg" height="170">
    <a href="#">    
        <svg display="none">
            <image src="img/fallback-logo.png" height="170" />
        </svg>
    </a>
</object>

CSS:

/*----------------------------------------*\
    SVG Pattern Styles
\*----------------------------------------*/
/*
    1. Make the link fit the content.
*/
.svg__link {
    display: inline-block; /* 1 */
}
