Page Layout - Article

There are currently two main page layouts available in Elementary - an 'article' layout where the focus of the page is text content, and a 'listing' layout, the purpose of which is to is to provide a list of articles.

This article page can contain any information or utility that you need. In addition there are some conventions in play:

Full-width container

By default there's a maximum width to elements at this level to prevent long lines of text and improve readability. To break out of this restraint simple wrap you content in a div.
