Hero

The Hero component can be in one of 2 places. There could be PAGE Hero (See top of this page for an example), or there could be a CONTENT Hero (see Article demp page) for an example.

The reason for the difference is about semantics. If a non-artcle page needs a Hero (like a listing page or form) then it makes sense this is part of the main page header. However if the pages is CONTENT page and had 'poster' image, it makes sense for this to be the Hero, but this means in needs to be part of the article itself, and the image should semantically appear under the HEADING so that screen readers will relate the image to the article and not skip over it.

Omit the small > details element if no info overlay is required.
