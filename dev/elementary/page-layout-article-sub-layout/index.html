This page contains an article 'sub-layout'. Sometimes it's necessary to a similar layout to the main page layout but within an article, using section elements rather than asides.

For example you may want a table of contents or important piece of information to appear about the main text, but if there's enough room it can nudge over to the side to make more space for the text (and save scrolling).

Similarly, there could be less crucial information that could sit underneath the main text but is useful to appear higher up in a sidebar if there's room.
