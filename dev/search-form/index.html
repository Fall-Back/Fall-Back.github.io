NOTES

The y="-10" on the <text> elements helps to shift the fallback text out of the viewport properly. In some browsers (iOS) the text bleeds through a little.

Opera Mini doesn't seem to allow the search field to shrink below some minimum width that seems to be related to the default size attribute setting. Using JS to target Opera Mini and add size="1" attribute to search fields allows it to shrink properly.
