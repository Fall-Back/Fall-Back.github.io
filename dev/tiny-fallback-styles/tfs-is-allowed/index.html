Example 8 - Some cards

Sample Card

Another Sample Card - this time with content

This card tells you something about something.

Another Sample Card - this time with an image

There is a source order problem here. Cards usually have their images above the heading, like it is here, but if the image is not presentational, it won't be accessible as part of the content the sits under the heading.

Currently the only way around this would be to include the image somewhere below the heading and use CSS to change the visual order, but in the Failed/No CSS scenario that's not possible.
